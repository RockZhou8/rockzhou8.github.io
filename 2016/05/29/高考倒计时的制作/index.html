高考倒计时的制作

发表于 2016-05-29   |   分类于 个人笔记   |  

一年一度的高考即将开始了，如果要在我们的网页上添加一个高考的倒计时，利用 JavaScript 就能很简单的做出来。

样例

现在我们有这样一个素材用来制作倒计时：

首先我利用 Photoshop 大致量出了我们要添加倒计时数字的位置，因为我们要在这个图片的“有”和“天”两字中间的空白处新建一个 span 用来显示这个倒计时的数字。

CSS 代码

给刚刚量出来的空白处的 span 命名 id 为 timeShow 写一下 CSS 代码：

#timeShow{
    position: absolute;
    top:85px;
    left:260px;
    font-size: 40px;
    line-height: 44px;
    color: white;
    align:center;
}

JavaScript 代码

window.onload = function(){
　　var gaokaotime= new Date("2016,6,7");  //定义高考时间     
　　var now = new Date();    //得到当前时间
　　var date = gaokaotime.getTime() - now.getTime();    //两日期的相差，得出的为毫秒
　　var time = Math.ceil(date/(1000 * 60 * 60 * 24));          //1000 * 60 * 60 * 24一天的毫秒数
     document.getElementById('timeShow').innerHTML = time;    
    }

高考时间的 new Date("2016,6,7") 括号中间为时间格式，也可以写作：

  • new Date(2016,5,7) 注意这里月份为 5 不是 6
  • new Date("2016/6/7")
  • new Date("6/7/2016") 或 new Date("June/7/2016")

结论

现在，我们利用 JavaScript 完成了这个高考倒计时的制作了。
