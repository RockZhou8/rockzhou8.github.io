CSS Filter Effects Blur Grayscale Brightness and More in CSS（译）

发表于 2016-07-17   |   分类于 个人译文   |  

原文：
CSS Filter Effects Blur, Grayscale, Brightness and More in CSS!


图为本文中一个 CSS 滤镜效果实例

滤镜是 SVG 规范原生的一部分。然而，当它们的用途变得更广泛时，W3C 也开始给 CSS 增加了一些常用的滤镜效果。CSS 滤镜非常强大又好用。你可以用它来模糊，变亮或饱和度调整图像。它们可以单独使用也可以和其它的滤镜组合使用。滤镜的 CSS 语法是这样的：

filter: <filter-function> [<filter-function>]* | none

现在开始，我来对这些滤镜做一下简要的介绍。

亮度

该滤镜控制图像的亮度。它接受大于或等于零的值作为它的参数。0％ 的值将变成黑色。100％ 的值则是原始图像。你可以指定值大于 100％ 得到更明亮的图像。比如 300％ 的值将使图像变成 3 倍亮：

img {      filter: brightness(300%);    }


原图

对比度

该滤镜控制图像的对比度。同上，它也接受大于或等于零的值。该滤镜控制 CSS 里图像的深色和浅色部分之间的差异。因此，0％ 的结果是一个灰色图像。设置对比度为 100％ 则是原始图像，超过之后将进一步提高图像的对比度：

img { filter: contrast(0%); }

Don’t Overthink It (Flexbox) Grids（译）

发表于 2016-07-03   |   分类于 个人译文   |  

原文：Don’t Overthink It (Flexbox) Grids

作者：Chirs Coyier

https://css-tricks.com/dont-overthink-flexbox-grids/

四年前，我贴出了一篇“Don’t Overthink It Grids”，它与一些人产生了共鸣。在当时，那可能是最高明的栅格系统。

那篇文章是我说道：“不要害怕，我们可以自己做一个栅格系统，不需要复杂的框架！”。它可能不是很花哨，不过你可以以一定比例的宽度漂浮一些元素。

现在，如果你学会玩转了 flexbox 布局，DIY 栅格系统就变得更容易了。

我现在经常用到的 HTML：

<div class="flex-grid">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>

CSS：

.flex-grid {
  display: flex;
}
.col {
  flex: 1;
}

这可以支持任意数量的列，它们都是自动相等的宽度，非常灵活！

想让它们在小屏设备上也有用？很简单：

CSS：

@media (max-width: 400px) {
  .flex-grid {
display: block;
  }
}

需要间隔？可以添加缩进和填充。我喜欢用对齐来创建列，比如：

.flex-grid-thirds {
  display: flex;
  justify-content: space-between;
}
.flex-grid-thirds .col {
  width: 32%;
}

如果你学会 Flexbox 的方法，也可以按需更改列，这可以保持更多的重要内容以及响应设计的调整。

回到顶部按钮的制作

发表于 2016-06-05   |   分类于 个人笔记   |  

我们经常会在浏览网页时，在比较长的页面上翻页之后，一旦想回到顶部去点导航栏或者想搜索些什么，可能需要用鼠标滚轮滚动多次才能回到顶部。比较方便的方法是在页面上浮动一个“回到顶部”的按钮，不管你在页面上哪个位置，点击它就能回到页面最顶部的位置了。

常规实现回到顶部的方法

最简单的方法无非是利用 HTML 里的锚点，给“回到顶部”按钮设置一个“#”的超链接，点击一下，就能很方便的回到顶部了。

<a href="#" id="baktop" title="回到顶部"></a>

带滚动效果的回到顶部

如果想点击“回到顶部”之后，页面能缓缓地回到顶部的位置，就需要给它设置 JavaScript 代码了，并且我们也能实现在页面一开始的第一屏不显示“回到顶部”按钮，只在页面滚动差不多第二页的时候才出现。

首先就不能把这个按钮设置成锚点了：

<a href="javascript:;" id="baktop" title="回到顶部"></a>

该按钮是这样一个图片，所以设置它的 CSS 样式：

#baktop {
width:40px;
height:40px;
position:fixed;
right:30px; 
display:block;
bottom:30px; 
background:url(../img/topbar.jpg) no-repeat left top;
}
#baktop:hover {
background:url(../img/topbar.jpg) no-repeat 0 -39px;
}

JavaScript 代码：

window.onload = function(){
var obtn = document.getElementById('baktop');
var clientHeight=document.documentElement.clientHeight;    //获取页面可视区的高度
var timer = null;
var isTop = true;

window.onscroll=function(){
  var osTop=document.documentElement.scrollTop||document.body.scrollTop;
if (osTop >= clientHeight){
    obtn.style.display="block"; //显示按钮
}else {
    obtn.style.display="none"; //隐藏按钮
}
if (!isTop){
    clearInterval(timer); //滚回到顶部时取消定时器
}
    isTop = false;  //页面滚动的情况
};
obtn.onclick = function(){
//设置定时器
timer = setInterval(function(){
    var osTop=document.documentElement.scrollTop||document.body.scrollTop;
    var ispeed = Math.floor(osTop / 6); //把速度定义成一个越来越小的值，就会有缓慢的效果
//获取滚动条距离顶部的高度
document.documentElement.scrollTop=document.body.scrollTop=osTop-ispeed;
isTop = true; //页面滚动到顶部时，取消定时器
if(osTop == 0){
    clearInterval(timer);
    }
},30);
}
    }

结论

利用 JavaScript，就可以制作出一个有很好的视觉效果的“回到顶部”了，比锚点那种直接的页面内跳转会显得更好。

高考倒计时的制作

发表于 2016-05-29   |   分类于 个人笔记   |  

一年一度的高考即将开始了，如果要在我们的网页上添加一个高考的倒计时，利用 JavaScript 就能很简单的做出来。

样例

现在我们有这样一个素材用来制作倒计时：

首先我利用 Photoshop 大致量出了我们要添加倒计时数字的位置，因为我们要在这个图片的“有”和“天”两字中间的空白处新建一个 span 用来显示这个倒计时的数字。

CSS 代码

给刚刚量出来的空白处的 span 命名 id 为 timeShow 写一下 CSS 代码：

#timeShow{
    position: absolute;
    top:85px;
    left:260px;
    font-size: 40px;
    line-height: 44px;
    color: white;
    align:center;
}

JavaScript 代码

window.onload = function(){
　　var gaokaotime= new Date("2016,6,7");  //定义高考时间     
　　var now = new Date();    //得到当前时间
　　var date = gaokaotime.getTime() - now.getTime();    //两日期的相差，得出的为毫秒
　　var time = Math.ceil(date/(1000 * 60 * 60 * 24));          //1000 * 60 * 60 * 24一天的毫秒数
     document.getElementById('timeShow').innerHTML = time;    
    }

高考时间的 new Date("2016,6,7") 括号中间为时间格式，也可以写作：

  • new Date(2016,5,7) 注意这里月份为 5 不是 6
  • new Date("2016/6/7")
  • new Date("6/7/2016") 或 new Date("June/7/2016")

结论

现在，我们利用 JavaScript 完成了这个高考倒计时的制作了。
